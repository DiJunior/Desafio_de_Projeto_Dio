(Voltar)

Head

(Voltar)

Head é a cabeça da página, a parte que não aparece para o usuário final, contudo é onde colocamos todas as informações para o nevegador. Informações importantes como a linguagem do site, os links de CSS e estilização da página, arquivos de fontes, entre outros.

Body

(Voltar)

Já o Body é a parte visível para o usuário. Nela que desenvolvemos toda a estrutura do site, como o título da página, conteúdo, imagens, tabelas, listas, etc.

H1 a H6

(Voltar)

As tags H1 até H6 são tags usadas para configurar a importância e o destaque daquele texto para o navegador e para o usuário. Neles você consegue coloca como por exemplo H1 que tem uma ênfase maior, e por isso será exibido em um tamanho maior no navegador. H2 com um tamanho menor e assim sucessivamente até o H6.

Para criar as tags com H você precisa colocar o texto dentro das tags, como por exemplo: < h1>Texto Aqui< /h1> (lembrando que é sem espaço nas tags, pois aqui é um exemplo)

Listas Ordenadas e Não Ordenadas

(Voltar)

As listas são tags que podemos usar para facilitar a criação de listas mesmo na página. Estas listas podem ser ordenadas, como por exemplo, item 1, item 2, etc. E as listas não ordenadas, onde o navegador exibe a lista sem marcações de números. Veja o exemplo de cada uma:

    Lista não ordenada
  • Item 1
  • Item 2
  • Item 3
    Lista Ordenada
  1. Item 1
  2. Item 2
  3. Item 3

Você pode perceber que na lista não ordenada, é criado uma bolinha na frente de cada item, já na ordenada, é criado um número que segue uma sequência lógica.

Para criar uma lista você utiliza a tag < ul > e fecha com < /ul>, assim como deve criar as tags < li> e fechar com < /li> em cada linha que irá inserir um item.

(Voltar)

Links de Navegabilidade

(Voltar)

Listas de navegabilidade é quando podemos utilizar marcarções dentro do próprio texto para ajudar na navegabilidade do site. Um exemplo é quando o texto é muito grande e você quer voltar ao topo, você utiliza como um link de atalho para voltar ao início do texto

Usamos id="nome" (identificadores) para marcar para onde queremos ir e usamos a tag < a> criar umm link para ligar os pontos. Por exemplo: < a href="id de referencia">
